El Protocolo de Contexto de Modelo (MCP) es un estándar abierto desarrollado por Anthropic 
que permite una integración estandarizada entre los modelos de IA y las herramientas externas. 
MCP actúa como un conector universal, permitiendo que los grandes modelos de lenguaje (LLM) 
interactúen dinámicamente con APIs, bases de datos y aplicaciones empresariales.

MCP sigue una arquitectura cliente-servidor donde los clientes MCP (como Claude Desktop) 
solicitan información y ejecutan tareas, mientras que los servidores MCP proporcionan acceso 
a herramientas y fuentes de datos externas.

Las ventajas principales de MCP incluyen:
- Integración estandarizada de la IA
- Flexibilidad para cambiar entre modelos
- Seguridad de datos
- Escalabilidad con múltiples transportes
